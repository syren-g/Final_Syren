# Final_Syren

Goal: Collect as many points as you can in a minute by eating (touching) the good (pink/golden) brains and avoiding the rotten (green) brains before they touch the ground! 

Use A and D keys to move Left and Right.

Press Space to Teleport a Small Distance.
